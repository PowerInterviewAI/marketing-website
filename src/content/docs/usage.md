# Usage Guide — Practical, Safe, Effective

This document covers practical usage: session setup, configuration, operating the
assistant, and privacy-minded tips to get meaningful improvements from your practice.

## Preparing a Good Session

- Choose a quiet, well-lit space and use a reliable headset to avoid echo.
- Use a neutral, professional-looking photo if you enable face-swap (optional).
- Prepare a short job description and your resume/CV to upload — the assistant uses
  this context to generate relevant questions and feedback.

## Configuration

1. Open **Profile → Settings**.
2. Set `Audio Input` and `Camera` to the devices you plan to use.
3. Select a language model preset or difficulty level (e.g., beginner, intermediate,
   senior engineer) depending on your target role.

## Running a Session

1. Click **Start Assistant** shortly before your mock interview begins. Credits
   are consumed while the assistant is running — stop it immediately after.
2. During the session, the assistant provides real-time hints and can optionally
   display suggested answers, follow-up questions, and timing cues.
3. After the session, export the transcript and AI feedback for review.

## Practical Advice

- Treat the assistant like a coach: try to answer fully before using suggestions.
- Use the playback and transcript to identify filler words, pacing, and repetition.
- Run focused practice sessions: e.g., 20 minutes for behavioral questions,
  20 minutes for technical whiteboarding.

## Troubleshooting

- If audio is choppy, reduce microphone gain or switch to a different device.
- If face-swap is unstable, use the default avatar and focus on voice/transcript feedback.

## Data & Privacy

Power Interview AI stores session data locally by default (check Settings). If you
enable cloud sync, review the retention policy and delete sessions you no longer
need. Avoid uploading sensitive PII in sample prompts.

This usage-focused guide is intentionally non-technical; it helps you get the most
from practice sessions without needing to read implementation details.

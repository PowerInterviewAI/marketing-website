# Integrations & Export

This product is primarily a desktop coaching application. The sections below
describe how to export data and integrate with other tools (not developer API
details). If you need programmatic access, contact support for an enterprise
integration plan.

## Exporting Session Data

- **Transcript**: Export a session transcript in plain text or JSON for study or
  import into note-taking tools.
- **Session Summary**: AI-generated highlights and suggested improvements.
- **Recordings**: If enabled, export audio (MP3) or video snippets for review.

Export steps:

1. Open the session history and select the session you want to export.
2. Click `Export` and choose file type (Transcript, Summary, Recording).

## Integrations

- **Notion / Obsidian / Evernote**: Export transcripts as Markdown to import into
  your knowledge base.
- **Calendar** (optional): Schedule mock interviews and attach a session template.
- **Enterprise**: Contact support to request SSO, dedicated storage, or an API
  integration.

If you need developer API access for automation, reach out to the team; it is
available under a commercial agreement and includes usage, quotas, and security
requirements.
